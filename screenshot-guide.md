# スクリーンショット撮影ガイド - Mozilla Store申請用

**最終更新**: 2025年1月9日

## 必要なスクリーンショット

Mozilla Add-ons Storeでの申請に必要なスクリーンショットを撮影してください。

### 1. 警告ダイアログのスクリーンショット

#### 撮影手順
1. Thunderbirdを起動
2. 新しいメールを作成
3. TO欄に複数のメールアドレスを入力
   - 例: `test1@example.com, test2@example.com`
4. 送信ボタンをクリック
5. 警告ダイアログが表示された瞬間を撮影

#### 撮影ポイント
- **警告メッセージ**: 「TO欄に複数のメールアドレスが検出されました！」が表示されている
- **送信停止**: 送信が停止されている状態
- **BCC推奨**: BCC欄の使用を推奨するメッセージ

#### ファイル名
`warning-dialog.png`

### 2. アドオン管理画面のスクリーンショット

#### 撮影手順
1. Thunderbirdを起動
2. `ツール` → `アドオン` を選択
3. インストール済みアドオンの一覧を表示
4. 「Thunderbird 誤送信防止アドオン」が表示されている状態を撮影

#### 撮影ポイント
- **アドオン名**: 「Thunderbird 誤送信防止アドオン」が表示
- **バージョン**: v1.1.4が表示
- **有効状態**: アドオンが有効になっている
- **説明**: アドオンの説明が表示されている

#### ファイル名
`addon-management.png`

### 3. 使用例のスクリーンショット

#### 撮影手順
1. Thunderbirdを起動
2. 新しいメールを作成
3. TO欄に複数のメールアドレスを入力
4. CC欄にも複数のメールアドレスを入力
5. 送信ボタンをクリック
6. 警告が表示された状態を撮影

#### 撮影ポイント
- **複数アドレス**: TO欄・CC欄に複数のアドレスが入力されている
- **警告表示**: システム通知・アラートが表示されている
- **送信停止**: 送信が停止されている状態

#### ファイル名
`usage-example.png`

## スクリーンショットの要件

### サイズ・形式
- **形式**: PNG形式
- **解像度**: 1920x1080以上推奨
- **ファイルサイズ**: 2MB以下

### 品質要件
- **鮮明**: 文字が読みやすい
- **適切なサイズ**: 重要な部分が十分に表示されている
- **プライバシー**: 個人情報（メールアドレス等）は適切に隠す

### 編集・加工
- **個人情報の隠蔽**: 実際のメールアドレスは `***@example.com` に置換
- **不要部分の削除**: 機密情報や不要な部分は削除
- **説明の追加**: 必要に応じて説明テキストを追加

## 撮影時の注意事項

### プライバシー保護
- **メールアドレス**: 実際のメールアドレスは使用しない
- **個人情報**: 名前・住所等の個人情報は表示しない
- **機密情報**: 企業情報・機密データは含めない

### 品質向上
- **明るさ**: 適切な明るさで撮影
- **コントラスト**: 文字が読みやすいコントラスト
- **フォーカス**: 重要な部分にフォーカスを合わせる

### 一貫性
- **デザイン**: 統一されたデザインで撮影
- **サイズ**: 同じサイズ・アスペクト比で撮影
- **スタイル**: 統一されたスタイルで撮影

## ファイル管理

### 保存場所
- **フォルダ**: `screenshots/` フォルダに保存
- **命名規則**: 分かりやすいファイル名を使用
- **バックアップ**: 複数の場所にバックアップ

### ファイル構成
```
screenshots/
├── warning-dialog.png      # 警告ダイアログ
├── addon-management.png    # アドオン管理画面
└── usage-example.png       # 使用例
```
